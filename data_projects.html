Films Industry Analysis

This is my first data analysis project consisting of three steps. I first go through the process of cleaning the data, which was aquired from kaggle, using the pandas library in Python. I then analyze the data using tools such as matplotlib, numpy, and seaborn to verify if there is correlation between variables that I am researching. Lastly I vizualize the data in an easy to understand format using Microsoft PowerBI.

Data Cleaning

Through the use of the Python libraries: pandas, and numpy, I am able to change various parts of the data that allow for ease of use during analysis. This ranges from removing rows with missing data, matching dates, and cleaning decimal values.

Data Analysis

Through the use of the Python libraries: matplotlib, and seaborn, I am able to create various charts and graphs that allow me to find correlations between specific column in my data.

Data vizualization

Using Microsoft PowerBI, I created a dashboard that highlights the correlation between various variables, and how they each can correalte with each other. This is done throught the use of various scatter plots, as well as a table.
